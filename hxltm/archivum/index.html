hxltmcli uses Python 3 and the reference version can be installed with hdp-toolchain. TL;DR: pip install hdp-toolchain. The result of the lastest hxltmcli --help. is on the bottom of this page.
The HXLTM core ontology cor.hxltm.yml that comes with hxltmcli can be customized with hxltmcli --archivum-configurationem path/to/mycopy.hxltm.yml.
For advanced hackers or (people helping others in middle of urgency and with basic knowledge of python), is possible copy only the single file hxltmcli.py, put on your executable path, and use immediately instead of hdp-toolchain version. The hard requirements are pip install libhxl langcodes pyyaml python-liquid

      #       <termEntry id="I18N_०१२३४५६७८९_〇一二三四五六七八九十百千万亿_-1+2/3*4_٩٨٧٦٥٤٣٢١٠_零壹贰叁肆伍陆柒捌玖拾佰仟萬億_I18N">
